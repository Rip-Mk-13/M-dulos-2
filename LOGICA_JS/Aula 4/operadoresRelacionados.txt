== Igualdade
0 == 0 -> True

Para analisar o valor e o tipo de dado, adicionamos mais um sinal de =
"0" == 0 -> True
"0" === 0 -> False

!= Diferente
0 != 0 -> False

Para analisar o valor e o tipo de dado, adicionamos mais um sinal de =
"0" != 0 -> False
"0" !== "1" -> False

> Maior

< Menor

2 >= 2 -> True
